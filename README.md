# DVLD - Driving & Vehicle Licensing Department System 🚗🪪

---

## 🌍 Overview | نظرة عامة
**[EN]** **DVLD** is a comprehensive desktop application developed using **C#** and **ADO.NET**.
The system is designed to automate and streamline the operations of a driving license department,
covering the entire process from person management to license issuance.

**[AR]** مشروع **DVLD** هو تطبيق مكتبي متكامل تم تطويره باستخدام لغة **C#** وتقنية **ADO.NET**. يهدف النظام إلى أتمتة وتنظيم العمليات الأساسية في دوائر ترخيص القيادة، حيث يغطي الدورة الكاملة ابتداءً من تسجيل البيانات وصولاً إلى إصدار الرخص.

---

## ✨ Core Features | الميزات الأساسية

### 👥 Management | الإدارة
*   **Person & User Management:** Centralized system for citizen data and secure employee access.
*   **إدارة الأشخاص والمستخدمين:** نظام مركزي لإدارة بيانات المواطنين وتأمين دخول الموظفين.

### 📄 Applications | الطلبات
*   **Workflow:** Handling new licenses, renewals, and replacements (lost/damaged).
*   **إدارة الطلبات:** معالجة طلبات الرخص الجديدة، التجديد، وإصدار بدل الفاقد أو التالف.

### 🧪 Testing System | منظومة الاختبارات
*   **Vision Test:** Physical eligibility check. (فحص النظر)
*   **Written Test:** Automated traffic rules exam. (الاختبار النظري)
*   **Practical Test:** Final driving evaluation. (الاختبار العملي)

### 🚫 Detain & Release | الحجز والإفراج
*   Manage traffic violations and license detentions.
*  وحدة خاصة لإدارة مخالفات المرور وحجز الرخص أو الإفراج عنها.  
---

## 🛠 Technical Stack | المواصفات التقنية

*   **Language:** C#
*   **Framework:** WinForms (.NET)
*   **Database:** SQL Server (ADO.NET)
*   **Architecture:** Multi-layered Architecture (3-Tier)

---

## ⚙️ Setup & How to Run | كيفية التشغيل

1.  **Database:** Execute the provided `.sql` script in **SQL Server Management Studio** to create the database.
2.  **Connection String:** Update the `ConnectionString` in your Data Access layer to match your local server.
3.  **Run:** Open the `.sln` file in **Visual Studio** and press `F5`.
